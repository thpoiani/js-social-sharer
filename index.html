JS Social Sharer

Avoid loading several JavaScript APIs for social sharing that can slow down your website

Social Networks

Some social networks have a Share Link Generator that enable the easily distribution of a share link.

Facebook

https://www.facebook.com/sharer/sharer.php?u=[URL]

Twitter

https://twitter.com/intent/tweet?url=[URL]

Google Plus

https://plus.google.com/share?url=[URL]

Only with a small amount of JavaScript, JS Social Sharer handle a hyperlink tag to create a generic sharing solution that don't require a powerful API from the social networks.

Usage

  1. Download JS Social Sharer
  2. Write your sharing links and include the script 
                                <a href="[URL]" data-sharer="facebook">FB Share</a>
    <a href="[URL]" data-sharer="twitter">Tweet</a>
    <a href="[URL]" data-sharer="google-plus">GPlus Share</a>

    <!-- add the script before </body> -->
    <script src="/path-to/js-social-sharer.js"></script>
  3. Try
